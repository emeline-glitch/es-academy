Workflow recommande
  1. Ouvre chaque visuel (lien ci-dessus)
  2. Cmd+P (Mac) / Ctrl+P (Windows)
  3. Destination : "Enregistrer au format PDF"
  4. Nomme le PDF avec le slug correspondant (ex: honoraires-d-agence.pdf)
  5. Place le PDF dans /public/ressources-formation/[module]/[sm]/
  6. Met a jour src/lib/ressources-manifest.ts avec le path .pdf au lieu de .jpg/.png
  7. Commit + push
